# WeDo2Lab

**Low-level LEGO WeDo 2.0 control over raw BLE**

---

## 🚀 Overview

WeDo2Lab is a collection of experiments, tools, and libraries for controlling **LEGO WeDo 2.0** hubs using **raw Bluetooth Low Energy (BLE)** commands.

Instead of relying on the official LEGO apps, this project focuses on:

* Direct BLE communication
* Reverse engineering the protocol
* Building custom control tools
* Learning how the hub really works under the hood

---

## ⚡ Features

* 🔗 Connect to WeDo 2.0 hubs via BLE
* ⚙️ Send raw commands to motors and LEDs
* 📡 Read sensor data
* 🧪 Experimental and reverse-engineered features
* 🧰 Minimal, hackable codebase

## 🧠 Background

The LEGO WeDo 2.0 hub communicates over BLE using a custom protocol.
This project explores that protocol by sending raw messages directly to the hub.

This allows:

* More control than the official app
* Integration with custom hardware/software
* Understanding of BLE device
